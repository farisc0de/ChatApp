# ChatApp

Simple PHP Chat System

## Features

1. Auth System
2. Emojis
3. Clean Interface
4. Responsive
5. Rooms
6. Encryption

## How to install

1. Import chat.sql
2. Run ```$ composer update ```
3. Update config.php [Keys are 128bit for IV and 256bit for the key]
5. Enjoy
